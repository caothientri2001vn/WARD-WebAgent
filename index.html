NeurIPS 2026 Submission

WARD: Web Agent Robust Defense against Prompt Injection

WARD is a practical guard framework for secure and efficient web agents. It combines large-scale multimodal training data, guard-targeted prompt injection training, and adaptive adversarial training to defend against malicious instructions embedded in HTML and rendered interfaces.

WARD-0.8b WARD-2b Code

Abstract

Existing guard models still suffer from limited generalization, high false positive rates on benign pages, deployment inefficiency, and vulnerability to adversarial or guard-targeted prompt injections. WARD addresses these issues with WARD-Base, WARD-PIG, and WARD-AAA, yielding strong generalization, low false positives, robust guard behavior, and efficient parallel execution with the agent.

Paper Summary

What The Paper Proposes

WARD builds a separate multimodal guard model that inspects the current screenshot, preprocessed HTML text, and the user intended task before the agent acts. The paper introduces three core components: WARD-Base, a large-scale benchmark and training dataset; WARD-PIG, a dataset for prompt injection attacks targeting the guard itself; and WARD-AAA, an adaptive adversarial training loop where attacker and guard co-evolve over repeated attack cycles.

Framework

WARD Pipeline

The paper combines data construction, reasoning supervision, guard-targeted training, and adaptive adversarial hardening.

Data

WARD-Base

WARD-Base contains 177,585 samples drawn from real web exploration and simulated high-risk platforms. It covers four injection-location groups, diverse native and overlay channels, six attack goal types, and a near-balanced benign-malicious split.

Guard Attacks

WARD-PIG

WARD-PIG explicitly targets the guard model itself by inserting misleading labels and structured outputs into the prompt context, training the guard to resist attacks on its own reasoning and decision boundary.

Adaptive Training

WARD-AAA

WARD-AAA uses an attacker with memory, a validator, and the guard in a closed loop. Successful and failed attacks are fed back into memory so the attacker generates progressively stronger prompt injections while the guard improves robustness.

Key Results

Detection, Utility, Robustness, Efficiency

Numbers below are taken from the paper text, tables, and accompanying artifacts.

Detection

On WARD-Test, the paper reports 99.33% accuracy and 99.34% F1 for the stronger released family, with near-perfect or perfect recall on Popup, EIA, VPI, and WASP.

Agent Defense

On VPI, WARD reduces downstream attack success rate to 0% across the reported browser-use and computer-use agent settings.

Utility

For GPT-4o WebArena agents, the paper reports false positive rates around 0.25% to 0.26% with negligible performance degradation, far below WebAgentGuard baselines.

PIG Robustness

WARD-0.8b and WARD-2b achieve 100% recall across HTML-only, screenshot-only, and both-modality Prompt Injection on Guard evaluations in the paper.

Adaptive Attacks

In the ablation table, adaptive adversarial training reduces sample success rate at try budget 10 to 5.62 for WARD-0.8b and 3.12 for WARD-2b.

Efficiency

WARD runs at about 2.24 to 2.45 seconds per step, versus 8.81 to 10.03 seconds for WebAgentGuard, enabling true parallel deployment without added latency.

Limitations

Scope

The paper focuses on prompt injections expressed through textual or visually interpretable content in HTML or rendered webpages. It is less targeted at pixel-level environmental attacks that rely on imperceptible perturbations and stronger attacker control over the rendering pipeline.
